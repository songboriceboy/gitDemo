java基础之对象的多态性

要求 掌握对象的向上转型及向下转型的使用
掌握对象转型的限制

多态性在两种主要体现
1.方法的重载和覆写
2.对象的多态

掌握了多态性之后 可以直接应用在抽象类和接口上


说了那么多 还是放一个实体理解下

package KownClass;

class A{					// 定义类A
	public void fun1(){		// 定义fun1()方法
		System.out.println("A --> public void fun1(){}") ;
	}
	public void fun2(){
		this.fun1() ;		// 调用fun1()方法
	}
};
class B extends A{
	public void fun1(){		// 此方法被子类覆写了
		System.out.println("B --> public void fun1(){}") ;
	}
	public void fun3(){
		System.out.println("B --> public void fun3(){}") ;
	}
};
public class test1{
	public static void main(String asrgs[]){
		/*子类对象--》父类对象 
		父类  父类对象=子类实例*/
		B b = new B() ;		// 实例化子类对象
		//因为b是继承了A 所以是A的子类
		A a = b ;			// 向上转型关系
		a.fun1() ;			// 此方法被子类覆写过
		a.fun2() ;
	}
};



对于以上的程序，是用过其子类进行父类对象的实例化操作的，则如果调用的方法被子类覆写过，
则肯定调用被读写过的方法
注意点：转型之后，因为操作的父类对象，所以无法找到在子类新定义的方法的


接下来是向下转型 将父类对象变为子类对象。向下转型需要强制手段

<span style="background-color: rgb(255, 255, 255);"></span>
package KownClass;


class A{					// 定义类A
	public void fun1(){		// 定义fun1()方法
		System.out.println("A --> public void fun1(){}") ;
	}
	public void fun2(){
		this.fun1() ;		// 调用fun1()方法
	}
};
class B extends A{
	public void fun1(){		// 此方法被子类覆写了
		System.out.println("B --> public void fun1(){}") ;
	}
	public void fun3(){
		System.out.println("B --> public void fun3(){}") ;
	}
};
public class test1{
	public static void main(String asrgs[]){
	/*	向下转型时，必须明确的指明要转型的子类类型，<span style="font-family: Arial, Helvetica, sans-serif;">*/</span>
		A a = new B() ;	// 向上转型关系
		//子类 子类对象=（子类）父类实例
		B b = (B)a ;		// 发生了向下转型关系
		b.fun1() ;
		b.fun2() ;
		b.fun3() ;
	}
};




因为有了向下转型所以fun3也可以被调用 同时如果要有向下转型就必须要有一个向上转型来建立关系 不然会报错


同时还要讲一下一个关键字instances of 可以判断是不是自己的子类 对于向下转型最好增加验证

class A{					// 定义类A
	public void fun1(){		// 定义fun1()方法
		System.out.println("A --> public void fun1(){}") ;
	}
	public void fun2(){
		this.fun1() ;		// 调用fun1()方法
	}
};
class B extends A{
	public void fun1(){		// 此方法被子类覆写了
		System.out.println("B --> public void fun1(){}") ;
	}
	public void fun3(){
		System.out.println("B --> public void fun3(){}") ;
	}
};
class C extends A{
	public void fun1(){		// 此方法被子类覆写了
		System.out.println("C --> public void fun1(){}") ;
	}
	public void fun5(){
		System.out.println("C --> public void fun5(){}") ;
	}
};
public class InstanceofDemo02{
	public static void main(String asrgs[]){
		fun(new B()) ;
		fun(new C()) ;
	}
	public static void fun(A a){
		a.fun1() ;
		if(a instanceof B){
			B b = (B) a ;
			b.fun3() ;
		}
		if(a instanceof C){
			C c = (C) a ;
			c.fun5() ;
		}
	}
};
